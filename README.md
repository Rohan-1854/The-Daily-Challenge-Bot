An interactive personal growth chatbot that generates personalized daily challenges across multiple categories such as Coding, Fitness, Learning, Productivity, and Mindfulness.
The bot encourages users to build better habits by completing small daily challenges.

🌐 Live Demo: https://daily-challenge-bot.netlify.app/

⚙️ Installation & Setup

To run this project locally:

1️⃣ Clone the repository
git clone https://github.com/Rohan-1854/Daily-Challenge-Bot.git
2️⃣ Navigate to the project folder
cd Daily-Challenge-Bot
3️⃣ Install dependencies
npm install
4️⃣ Start the development server
npm run dev

The app will run on:

http://localhost:5173
🎯 How It Works

The chatbot greets the user.

The user selects an interest category.

The system generates a challenge related to that category.

The user can complete the challenge and come back for new ones daily.

This design encourages consistent habit building and personal development.
